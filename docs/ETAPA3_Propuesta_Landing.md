# ETAPA 3 — Propuesta Landing Page

## Diseño propuesto

**Estilo:** Profesional, limpio, moderna. Enfoque B2B (distribuidora de alimentos para mascotas).

### Elementos
- **Hero:** Título + subtítulo breve sobre auditorías comerciales
- **Ubicación:** Villa María, Córdoba
- **Botones:** "Registrarse" (primer admin) y "Iniciar sesión"
- **Colores:** Paleta sobria – azul/dark para seriedad, acento cálido (ámbar) para calidez
- **Tipografía:** Clara y legible (Inter o similar)
- **Responsive:** PC y móvil

### CTA principales
1. **Registrarse** → `/registro` (para el primer administrador)
2. **Iniciar sesión** → `/login` (usuarios existentes)
